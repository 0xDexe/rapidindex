Key Features of this Base Parser:

Abstract Base Class - Forces subclasses to implement parsing logic
 Type Hints - Full type annotations throughout
 Comprehensive Validation - File existence, format, size checks
 Error Handling - Custom exceptions with detailed messages
 Logging - Structured logging with loguru
 Metrics Collection - Tracks parsing performance
 Configuration - Pydantic-based config with validation
 Post-processing - Common cleanup and metadata addition
 Extensibility - Helper methods for subclasses to use
 Documentation - Comprehensive docstrings
